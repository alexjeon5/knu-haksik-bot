# KNU HakSik Bot
Kyungpook National Univeristy School Meal Bot

- 경북대학교 학식 알리미 봇
- 봇 추가하기 : https://t.me/KNU_HaksikBot

## 현재 구현된 기능 설명 (2026-04-06 기준)
### 주요 기능
- **실시간 식단 조회**: 경북대학교 내 주요 식당의 오늘의 식단을 텔레그램을 통해 즉시 확인할 수 있습니다.
- **점심/저녁 분리 출력**: 기본적으로 점심 식단을 보여주며, `저녁` 키워드를 조합해 석식 메뉴만 따로 조회할 수 있습니다.
- **자연스러운 단축어(별명) 지원**: 길고 복잡한 공식 이름 대신 `정센`, `공식당`, `첨성` 등 학생들이 자주 쓰는 줄임말로 편하게 검색할 수 있습니다.
- **친절한 사용 안내**: `/start` 또는 `/help` 명령어 입력 시 직관적인 사용법 안내와 전용 키보드(버튼)를 제공합니다.
- **자동 식단 업데이트**: 매주 월요일 오전 6시, 백그라운드 스케줄러가 자동으로 한 주간의 식단 데이터를 크롤링하여 갱신합니다.

### 지원 식당 및 단축어 목록
기본 공식 명칭 외에도 괄호 안의 단축어를 입력하여 식단을 조회할 수 있습니다.
* **정보센터식당** (`정센`, `정보센터`)
* **복지관 교직원식당** (`복지관교직원`)
* **카페테리아 첨성** (`복지관`, `첨성`, `카페테리아`)
* **GP감꽃식당** (`감꽃`, `감꽃식당`, `글플`)
* **공식당 교직원식당** (`공교`, `공식당교직원`, `공대교직원`)
* **공식당 학생식당** (`공학`, `공식당학생`, `공식당`, `공대`)

### 사용 방법
봇 대화창에 아래의 명령어 및 텍스트를 입력해 보세요.

* `/start` 또는 `/help`: 봇 사용 설명서를 확인하고 '급식' 기본 버튼을 띄웁니다.
* `급식`: 어떤 식당을 조회할지 선택할 수 있는 전체 버튼 목록을 화면에 표시합니다.
* `[식당 이름 또는 단축어]`: 해당 식당의 **점심** 식단을 보여줍니다.
    * *(예시: `공학`, `첨성`, `공식당`)*
* `저녁 [식당 이름 또는 단축어]`: 해당 식당의 **저녁** 식단을 보여줍니다.
    * *(예시: `저녁 정센`, `저녁 공교`)*
 
## 추후 구현 예정인 기능
- 자동 알리미는 날짜별로 세분화 해서 설정할 수 있도록 계획하고 있음.
- 추후 디스코드나 카카오톡 등의 조금 더 친숙한 플랫폼으로 확장 할 계획 있음.
