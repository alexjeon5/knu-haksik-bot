# KNU HakSik Bot
Kyungpook National University School Meal Bot

경북대학교 학식 알리미 봇  
봇 추가 링크 : https://t.me/KNU_HaksikBot

## 현재 구현된 주요 기능
1. 실시간 식단 조회 : 경북대학교 내 주요 식당의 오늘의 식단을 즉시 확인 가능합니다.
2. 점심 및 저녁 분리 출력 : '점심' 또는 '저녁' 키워드를 조합하여 원하는 시간대의 메뉴를 조회합니다.
3. 다양한 단축어 지원 : 정센, 공학, 첨성 등 학생들이 자주 쓰는 줄임말로 검색이 가능합니다.
4. 내일 식단 및 주말 처리 : '내일' 키워드 사용 시 다음 날 식단을 보여주며, 금요일 이후 사용 시 다음 주 월요일 식단으로 자동 매칭됩니다.
5. 자동 예약 알림 시스템 : 사용자가 지정한 요일, 시간, 식당에 맞춰 매일 학식 정보를 자동 전송합니다.
6. 주간 데이터 자동 갱신 : 매주 월요일 오전 6시, 최신 식단 데이터를 자동으로 크롤링하여 업데이트합니다.

## 사용 방법
봇 대화창에 아래의 텍스트를 입력하세요.

- /start 또는 /help : 봇 사용 설명서를 확인하고 식당 선택 버튼을 출력합니다.
- 점심 또는 학식 : 식당을 선택할 수 있는 전체 버튼 목록을 표시합니다.
- [식당 이름 또는 단축어] : 해당 식당의 점심 식단을 보여줍니다.
- 저녁 [식당 이름 또는 단축어] : 해당 식당의 저녁 식단을 보여줍니다.
- 내일 [식당 이름 또는 단축어] : 해당 식당의 다음 영업일 식단을 보여줍니다.
- 예약 : 자동 알림 설정 메뉴(신규 생성, 수정, 취소)로 진입합니다.
  
`/start` 와 `/help` 를 제외한 모든 명령어는 `/`없이 사용할 수 있습니다.

## 지원 식당 및 단축어 목록
- 정보센터식당 (정센, 정보센터)
- 복지관 교직원식당 (복지관교직원)
- 카페테리아 첨성 (복지관, 첨성, 카페테리아)
- GP감꽃식당 (감꽃, 감꽃식당, 글플)
- 공식당 교직원식당 (공교, 공식당교직원, 공대교직원)
- 공식당 학생식당 (공식당, 공학, 공식당학생, 공대)

## 추후 구현 예정 사항
- 디스코드 등 타 플랫폼 서비스 확장 고려
- 본인 사이트에서 조회할 수 있도록 학식 데이터 저장 방식 변화
- 사용자 의견 수렴 후 조정사항 적용

- 학식 자동 알리미가 필요해서 제작하게 됨.
